GIT HUB COMANDOS


git init =  inicia o crea un nuevo repositorio local

git status=  ver el estado de los archivos en el repositorio

git add (nombre del archivo)=  manda un archivo especifico al area de prueba
git add -A = manda todos los archivos al area de prueba (mas recomendable que git add . porque efectua todos los cambios incluso los archivos eliminados)

git commit -m "un mensaje describiendo los cambios hechos" = sirve para subir en el repositorio local los cambios hechos y si da el caso volver a dicho archivo despues de alguna actualizacion erronea; a eso se le llama un commit

git branch -M main =    cambiar el nombre de la rama master a main

git remote add origin (url de el repositorio) =   designa una repositorio remoto al local

git push -u origin main  =  finalmente este comando los sube al repositorio remoto
